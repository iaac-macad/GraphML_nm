# GraphML_nm

Graph ML coursework repository with notebooks and supporting datasets for graph-based modeling tasks.

## Repository Structure

- `GML_notebooks_folder/`: lesson notebooks and supporting files
- `HW01/`: homework notebook and Rhino working files

## Contents

- Graph representation concepts (primal/dual, metric/topological, adjacency/access)
- Spatial intelligence notebooks
- IFC semantic and spatial relationship notebooks
- Dataset and feature engineering notebooks
- Classification and regression notebooks for Graph ML

## Datasets

Datasets are available under:

- `GML_notebooks_folder/Supporting Files/dataset_classification/`
- `GML_notebooks_folder/Supporting Files/dataset_regression/`

Each dataset folder includes:

- `nodes.csv`
- `edges.csv`
- `graphs.csv`
- `meta.yaml`

## Environment

This project is typically used with a local Python virtual environment (`.gmlenv`) and Jupyter notebooks.
